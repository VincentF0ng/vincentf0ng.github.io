nodemailer 介绍

Nodemailer 是 Node.js 一个开源插件，可以在 Node 环境中轻松实现发送电子邮件。官方文档


要求

Node v6.0.0 或更新版本。如果 Nodemailer 中使用async..await，你需要至少有 Node v8.0.0 。

基础用法

使用QQ邮箱发送为例：

  • host：smtp.qq.com
  • port：465
  • secure：true
  • auth.user：邮箱地址
  • auth.pass：邮箱授权码 （登录邮箱后台获取）
